Попапы (модальные окна)

Для поапов используелся плагин Magnific Popup. Список базовых состояние попапов собран для задачи №73513 в Тимере.

Что есть из попапов

  1. Без описания
  2. Без кнопок и крестика
  3. Обычный
  4. С терминальной зоной и кнопкой справа
  5. С терминальной зоной и ссылкой справа
  6. С зоной скролинга между шапкой и подвалом

Сделанное по макетам дизайнера

  1. Выберите способ оплаты
  2. Вход/регистраниция/восстановление

Пример кода для вызова поапа. Подробности — в документации.

$('.open-popup').magnificPopup({
  type: 'inline',

  fixedContentPos: fatruelse,
  fixedBgPos: true,

  overflowY: 'auto',

  closeBtnInside: true,
  preloader: false,

  midClick: true,
  removalDelay: 300,
  mainClass: 'my-mfp-zoom-in'
});

Сначала необходимо зайти в «Проекты в системе», далее рядом с названием проекта, которое вы хотите поменять, кликаете ссылку «ПРОФИЛЬ». В открывшемся окне можно сменить название проекта. После внесения изменений не забудьте «СОХРАНИТЬ». Обращаем ваше внимание, что при смене названия проекта его сортировка не меняется.

1. Откройте пункт меню «Проекты в системе» (располагается в левой части сайта)

2. Откроется список проектов, рядом с каждым названием проекта есть ссылка «профиль». Откройте профиль нужного проекта.

3. В открывшемся окне найдите поле «Название сайта», в этом поле впишите новое название проекта;

Обращаем ваше внимание, что при смене названия проекта его позиция по отношению к другим проектам в системе (если они есть) не меняется.
